T("biquad")

AR Biquad Filter

Description

フィルター

Properties

  • type (String)
    • Filter Types を参照してください
  • freq, cutoff (T-Object)
    • カットオフ周波数
  • Q (T-Object)
    • Q
  • gain (T-Object)
    • ゲイン

Methods

  • plot(opts)
    • フィルタ特性を描画します

Filter Types

type プロパティで指定できるフィルターの種類です。エイリアスが設定されているので T("lowpass") や T("lpf") のように直接生成することもできます。

Low-Pass Filter

alias: lpf

High-Pass Filter

alias: hpf

Band-Pass Filter

alias: bpf

HighShelf Filter

Peaking Filter

Notch Filter

alias: bef, brf

alias: apf
